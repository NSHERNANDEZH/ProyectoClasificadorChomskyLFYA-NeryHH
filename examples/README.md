# Ejemplos de Gramáticas

Este directorio contiene ejemplos de gramáticas para diferentes tipos de la Jerarquía de Chomsky.

## Archivos

- `example_grammars.txt`: Colección de gramáticas de ejemplo organizadas por tipo

## Uso

Puedes copiar y pegar estas gramáticas en la interfaz de Chomsky Classifier AI para analizarlas.

## Tipos Incluidos

- **Tipo 3**: Gramáticas regulares (derecha e izquierda)
- **Tipo 2**: Gramáticas libres de contexto
- **Tipo 1**: Gramáticas sensibles al contexto
- **Tipo 0**: Gramáticas recursivamente enumerables
- **Casos especiales**: Gramáticas ambiguas, con recursión a izquierda, etc.
